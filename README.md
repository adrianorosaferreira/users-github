Repositorios de Usuarios do GitHub
=============================

Este projeto exibe os repositorios de um usuario do GitHub utilizando-se das seguintes tecnologias:
 - Framework [AngularJS] 
 - Biblioteca jQuery
 - Framework front-end Bootstrap
 - API disponibilizada pelo GitHub.
